# R3F WebGPU + TSL Template

This is a template for creating Three.js WebGPU projects with React Three Fiber and TSL.

## Getting Started

To get started, run the following commands:

```bash
bun install
bun run dev
```

## Features

- React Three Fiber
- WebGPU
- TSL
- ESLint
- Prettier
- Bun
- Vite

- Noises (simplex-noise/open-simplex-noise)
